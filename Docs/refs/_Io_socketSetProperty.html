_Io_socketSetProperty

Server-side only. Bind a custom property to a socket, which can be used later on

_Io_socketSetProperty(Const $socket, $sProp, $value)

Return Value

True if successfull, @error if error.

Remarks

the props `socket`, `ip` and `date` is reserved. @Error 1 = reserved prop name used. @Error 2 = invalid property name. Only a-zA-Z 0-9 and _ is allowed.
