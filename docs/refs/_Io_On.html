_Io_On

Binds an event

_Io_On(Const $sEventName, Const $fCallback = Null, $socket = $g__io_mySocket)

Return Value

None

Remarks

If $fCallback is set to null, the function will assume the prefix "_On_" is applied. Eg (_Io_On('test') will look for "Func _On_Test(...)" etc
