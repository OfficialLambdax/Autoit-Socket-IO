_Io_socketGetProperty

Server-side only. Retrieves information about the socket.

_Io_socketGetProperty(Const $socket, $sProp = Default, $default = Null)

Return Value

Mixed

Remarks

If $sProp is set to `Default` then an array containing two elements will be returned.
